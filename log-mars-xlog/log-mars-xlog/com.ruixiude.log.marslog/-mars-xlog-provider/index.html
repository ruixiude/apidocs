日志服务提供者 - Mars Xlog 实现

  • namePrefix 日志文件前缀

  • logPath 日志文件存放目录

  • cachePath 日志缓存存放目录，缓存目录类似日志目录，要单独目录，防止清理逻辑误删其他文件，而且缓存目录一定要是data下的私有目录，可以使用 /data/data/packagename/files/xlog/

  • debug 是否开启 Debug 模式，默认关闭
